# A classic game Snake
* Name:贪吃蛇
* Programming language:C
* version:1.3
* Platform:windows(based on some librarys)

> This is a **console Application**.There are **three leves** in the game and you can choose the level in the beginning .This game also provide the funcation that you can **pause** or **save the game** or **load profile** or **ranking list** so on.For further details,you can read the source code or build this project.

****

* 控制台程序贪吃蛇
* 基于C语言
* 包含食物（长度加一），炸弹（长度减半），毒草（长度减一）道具。
* 三个关卡
* 实现简单的排行榜，暂停，存档功能。
